# Documentation

## Overview

Nadi for Yii 3 is a monitoring SDK that integrates with Yii 3 applications to capture exceptions,
HTTP requests, slow database queries, and application performance metrics.
It supports multiple transport drivers and OpenTelemetry trace context propagation.
